第八节 变量操作函数

1 empty：检测变量是否为空

bool empty ( mixed var)

当var为"", 0, "0", NULL, FALSE, array(), var $var;等时，返回TRUE，否则返回FALSE。

2 gettype： 取得变量的类型名称

string gettype ( mixed var)

返回变量的类型名称，可能的值有：boolean,integer,double,string,array,object,resource,NULL,unknown。

3 intval：取得变量的整数值

int intval ( mixed var [, int base])

返回var以base指定的进制表示的整数值。Var是标量，intval不能用在array 或object类型数据上。

4 floatval: 取得变量的浮点数数值

float floatval ( mixed var)

返回var的浮点数数值。Var是标量，intval不能用在array 或object类型数据上。

5 strval：取得变量的字符串数值

string strval ( mixed var)

返回var的字符串数值。Var是标量，intval不能用在array 或object类型数据上。

6 isset：检测变量是否存在

bool isset ( mixed var [, mixed var [, ...]])

若变量var存在，返回TRUE，否则返回FALSE。

7 unset：删除已有变量

void unset ( mixed var [, mixed var [, mixed ...]])

该函数删除指定的变量。例如： 

  <?php
    // destroy a single variable
  unset($foo);

  // destroy a single element of an array
   unset($bar['quux']);

  // destroy more than one variable
  unset($foo1, $foo2, $foo3);
  ?>

8 var_dump：显示变量的相关信息

void var_dump ( mixed expression [, mixed expression [, ...]])

该函数显示变量的结构形式，包括其数据类型和数值。例如： 

<?php
$a = array (1, 2, array ("a", "b", "c"));
var_dump($a);
?>
输出：
array(3) {
  [0]=>
  int(1)
  [1]=>
  int(2)
  [2]=>
  array(3) {
    [0]=>
    string(1) "a"
    [1]=>
    string(1) "b"
    [2]=>
    string(1) "c"
  }
}
又例如：
<?php

$b = 3.1;
$c = true;
var_dump($b, $c);

?>
输出：
float(3.1)
bool(true)
