列表元素

列表元素简介

HTML 提供了下列几种列表元素：

  • 无序列表.
  • 有序列表.
  • 定义列表.

无序列表（UL，unordered lists），有序列表（OL， Ordered lists）, 列表项目 ( LI )

无序列表和有序列表在表现形式上很相似，区别在于有序列表 的列表项目前面有顺序标记，如数字或罗马字符。两者的列表项目 都是用LI标记元素表示，其结束标记可以省略。

下面的例子表示出列表元素的基本的结构

<UL>
   <LI> ... first list item...
   <LI> ... second list item...
   ...
</UL>

列表元素可以嵌套使用:

<UL>
     <LI> ... Level one, number one...
     <OL>
        <LI> ... Level two, number one...
        <LI> ... Level two, number two...
        <OL start="10">
           <LI> ... Level three, number one...
        </OL>
        <LI> ... Level two, number three...
     </OL>
     <LI> ... Level one, number two...
</UL>

编号的详细说明.在有序列表中， 作者可以重新设置某个列表项目的value属性值，从而改变其 序号。其后的项目就按改变后的序号排列。例如:

<ol>
<li value="30"> makes this list item number 30.
<li value="40"> makes this list item number 40.
<li> makes this list item number 41.
</ol>

定义列表: DL, DT, 和 DD 元素

定义列表包含两个部分: 主题词及其描述. 主题词用DT元素表示， 属于行内元素，描述部分用DD元素表示，属于块级元素。

一个例子:


<DL>
  <DT>Dweeb
  <DD>young excitable person who may mature
    into a <EM>Nerd</EM> or <EM>Geek</EM>

  <DT>Hacker
  <DD>a clever programmer

  <DT>Nerd
  <DD>technically bright but socially inept person

</DL>

下面是一个多重主题词和多个描述的例子：

<DL>
   <DT>Center
   <DT>Centre
   <DD> A point equidistant from all points
              on the surface of a sphere.
   <DD> In some field sports, the player who
              holds the middle position on the field, court,
              or forward line.
</DL>

浏览器可以不管DD标记是否紧跟DT标记之后，都能以缩进的方式显示DD标记所包含 的内容。

作者可以
<DL>
   <DD> 通过设置其type属性得到不同的视觉表现。对于UL元素，
   其type属性的可能取值有disc, square, 以及circle。
   它们对大小写不敏感。
   </DD>

</DL>

DL元素的另一个实际应用实在表示对话内容上, 可以用DT表示 说话者，用DD表示说话的内容。

列表的视觉呈现

对于OL和UL元素，都可以通过设置其type属性得到不同的视觉表现。

对于UL元素，其type属性的可能取值有disc, square, 以及circle。 它们对大小写不敏感。

对于OL元素，其可能的取值见下表，它们是大小写敏感的。

Type Numbering style
1 arabic numbers 1, 2, 3, ...
a lower alpha a, b, c, ...
A upper alpha A, B, C, ...
i lower roman i, ii, iii, ...
I upper roman I, II, III, ...
