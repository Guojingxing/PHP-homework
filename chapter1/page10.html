多窗口页面(Frames)

基本语法

<frameset> ... </frameset>
<frame src="url">
<noframes> ... </noframes>

在 <noframes> 标记后的文字将只出现在不支持 FRAMES 的浏览器中。

        <HTML>
        <HEAD>
        </HEAD>
        <FRAMESET>
             <FRAME SRC="url">
             <NOFRAMES> ... </NOFRAMES>
        </FRAMESET>
        </HTML>

各窗口的尺寸设置

<frameset cols=#>

纵向排列多个窗口： 

<frameset cols="30%,20%,50%">
<frame src="left.html">
<frame src="middle.html">
<frame src="right.html">
</frameset>

</frameset>

 浮动窗口(Floating Frame)






<iframe src=# name=##> ... </iframe> 

#=初始页面的 URL

##=窗口标识(Frame Name)(之后可对此标识进行各窗口间相互操作)

... = 此处文字将只出现在不支持 FRAMES 的浏览器中。


浮动窗口相当于在一行文本或一个文本块中插入一个窗口。





<center>
<iframe src="left.html" name="window">
   Here is a Floating Frame
</iframe>
<br><br>
<a href="middle.html" target="window">中间页面</A><BR>
<a href="top.html" target="window">上部页面</A><BR>
<a href="bottom.html" target="window">底部页面</A><BR>
</center>
